My Reflection

These are the challenges that I have faced while creating the Website and how I overcame them

  1. 1.When designing a website, it is crucial to keep the design very basic such as simple and appropriate colors while keeping in mind that it matches the theme and the content.
  2. 2.Bad value characters. Bad value for "mail" for attribute type on element input. W3School HTML validator have detected this error meaning "mail" should have been written as "email".
  3. 3.No space allowed. While writing values, there are null values that were not filled. Using W3School validator, the error is detected and a positive integer value was suggested to insert in the null space.
  4. 4.The website should be responsive so that it can be run on different platforms such as mobile phones, laptops and PCs.
  5. 5.While I was working on my website, I faced challenges such as bugs where because of a missing item, the whole code was not working. I had to spend all night to re check all the codes and then find the mistake.
  6. 6.Usually, when inserting pictures in my website, I misjudged and placed it in another folder.
  7. 7.Finally, I did remember afterwards to download the images into the assessment folder for the website.
  8. 8.Moreover, I had issues with inserting my portfolio files and the ZIP files also as there were too many and were not well arranged. I had to arrange them by order and insert them into my webpage. I had issues with designing the website as the interface was a bit difficult to handle since it had to match with my picture on the home page. I had to design and inspect the website each time to eliminate bugs and to build the website the way it is now.
  9. 9.The W3Schools website has helped me to learn new things about my errors and how to manipulate the layout of my website.
